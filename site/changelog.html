Release notes

Glass has no tagged releases yet — these are the shipped milestones so far, written for someone deciding whether the product is alive and worth trying. It has no marketing department; each entry links to the merged proof.

2026-07-04 - fleet wall

Aggregated fleet stage

  • Every live agent lane now rolls up into one fleet wall: one card per session, its newest surface, sorted by recency, refreshed without a page reload.
  • Clicking a lane's card drills into that one session's full activity, with a link back to the aggregate view.

Evidence: merged, redeployed, and verified live.

2026-07-04 - branding + identity

The stage wears its own mark

  • The header carries the Glass wordmark and its mirror icon — no leftover branding from the interim viewer it replaced.
  • Repeated posts from the same agent now collapse into one sidebar identity instead of one entry per session.

2026-07-04 - verified-live onboarding

Push-button deploy, provably live

  • glass doctor checks a running service end to end: reachable, backed by the right SQLite database, and able to publish and read back a disposable probe post.
  • Documented the tailnet deployment contract, including rollback.

2026-07-04 - MVP

The live stage, first cut

  • One Rust service where agents publish versioned, typed surfaces — markdown, diff, html, image, terminal, mermaid, json, code — grouped into per-agent sessions.
  • Sandboxed rendering with per-response CSP, content-addressed assets, and a one-way publish contract with no operator reply channel back to the producing agent.
